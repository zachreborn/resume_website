.About This Website

This website was created using HTML and CSS entirely by me with a little help from GitHub CoPilot. The website is hosted on AWS utilizing Route53, CloudFront, and S3 services. The repositories for the website and the cloud environment are hosted on GitHub. The cloud environment was created and is maintained using terraform. The website is automatically deployed with GitHub Actions workflows. The website itself was built to be responsive and works on devices with all screen dimensions.
